# Triangular-Square
Design of an electronic circuit that converts a triangular wave input signal into a square wave output using comparator/op-amp principles.
## Objective
The main objective of this project is to generate a square wave signal from a triangular input signal by using comparator principles in electronic circuits.

## Working Principle
A comparator compares the triangular input signal with a reference voltage.  
- When the input voltage is greater than the reference voltage, the output becomes HIGH.  
- When the input voltage is lower than the reference voltage, the output becomes LOW.  

This switching action produces a square wave output corresponding to the triangular input waveform.

## Components Used
- Operational Amplifier (Op-Amp)
- Resistors
- Function Generator (Triangular signal input)
- Power Supply
- Oscilloscope

## Applications
- Signal processing
- Waveform conversion
- Pulse generation
- Digital electronics circuits

## Conclusion
The circuit successfully converts a triangular waveform into a square waveform using a comparator-based design. This demonstrates the practical application of operational amplifiers in signal conditioning and waveform conversion.
